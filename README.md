# SafeRoad AI+

## AI-Powered Road Risk Detection & Emergency Response System

SafeRoad AI+ is an intelligent road safety platform developed for the IIT Madras Road Safety Hackathon.

The system combines:
- AI-based pothole detection
- Road risk analysis
- Hazard mapping
- Emergency SOS response

to improve road safety and reduce accident risks.

---

## Features

### AI Road Damage Detection
Detects:
- potholes
- cracks
- waterlogging

using computer vision models.

### Road Risk Score
Assigns:
- Safe
- Moderate Risk
- High Risk

based on road conditions.

### Hazard Heatmap
Displays dangerous roads and accident-prone areas.

### Emergency SOS
Provides instant GPS-based emergency alerts.

### Authority Dashboard
Allows authorities to monitor and resolve complaints.
